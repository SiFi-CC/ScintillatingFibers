Detailed Description

Class to perform background subtraction from charge spectra. It is possible to subtract background from 511 keV ans 1270 keV peak, depending on peak ID given in the constructor. Backround is described as exponential function. Class finds peak position and sigma along with their errors and creates histogram with background-subtracted peak.
