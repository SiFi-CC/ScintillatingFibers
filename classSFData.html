Detailed Description

Class to access experiemntal data. It is possible to get single histograms and vectors of histograms of requested type. Single signals and averaged signals are also possible to access. Cutting functionality for all has been implemented.

◆ InterpretCut()

bool SFData::InterpretCut ( DDSignal *  sig,
TString  cut 
)
private

Parses given cut and checks if signal fulfills conditions specified by it.

Parameters
sig- currently analyzed signal, as read from the tree
cut- a logic cut to select specific signals. The following syntax of the cut is acceptable:
  • inequality signs: '<' and '>'
  • single expressions, e.g. "fAmp>50", "ch_0.fPE>10", "fT0<100"
  • double expressions with &&, e.g. "fPE>10 && fPE<100", "ch_0.fT0>0 && ch_0.fT0<500"

If empty cut is passed returns true.

◆ SetDetails()

Sets all details of selected experimental series. If default constructor was used, this function needs to be called explicitly with the number of of requested series as an argument. The following attributes are set within this function:

  • fiber type
  • measurement description
  • positions of radioactive source for all measurements
  • names of measurements in this series
