Detailed Description

Class designed to locate 511 keV in the recorded spectrum. With the provided set of functions it is possible to find the peak using ROOT's TSpectrum (SFPeakFinder::FindPeakSpectrum()), by fitting sum of functions describing shape of spectrum (SFPeakFinder::FindPeakFit()) and by backgournd subtraction (SFPeakFinder::FindPeakNoBackground()). All of the methods provide information about peak position and sigma along with their uncertainties. The latter method creates histogram with background subtracted 511 keV peak. Additionally, it is possible to get peak range.

Testing mode implemented for some methods in order to check fitting and background subtraction quality.
