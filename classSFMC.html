Detailed Description

Class to access montecarlo data. It is possible to get single histograms and vectors of histograms of requested simulated data.

◆ SetDetails()

bool SFMC::SetDetails ( int  seriesNo)

Sets all details of selected simulated series. If default constructor was used, this function needs to be called explicitly with the number of of requested series as an argument. The following attributes are set within this function:

  • number of measurements in the series
  • fiber type
  • measurement description
  • simulated positions of the radioactive source for all measurements
  • names of simulated measurements in this series
